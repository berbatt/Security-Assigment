# Corona Statistics Virus

A virus that spreads through the python files which are currently at the same or under the directory of the virus.
It copies encrypted version of itself to the other python files and executes the payload.

In the payload part it uses a Coronavirus Tracker(https://github.com/sagarkarira/coronavirus-tracker-cli#----coronavirus-tracker-cli)
to show the statistics about the top 10 countries with the Corona virus.
